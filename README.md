# Getting Started

    1. Install Dart SDK
    2. Clone the repo
    3. Run `dart pub get` to install dependencies
    4. Run `dart run build_runner build` to generate code
    5. Run `dart run bin/main.dart` to start the server
    6. Open `http://localhost:8080` in your browser
